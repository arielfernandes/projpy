# projpy

A simple shell script to create a python project structure
  
# Structure
 ~~~
 myProj
├── docs
├── myProj
│   ├── app.py
│   └── __init__.py
├── README.md
└── tests
~~~

# Usage
Usage:  -c create_folders -a "author" -e venv
	-c Name of the root directory that starts the project
	-a Author
	-m Email
	-e Start Venv
